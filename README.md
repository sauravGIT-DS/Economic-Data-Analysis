
# Cost of Living Index Analysis (2022)

## Project Overview
This project analyzes the **Cost of Living Index 2022** dataset to understand global living expenses,
purchasing power, and affordability across countries using Exploratory Data Analysis (EDA).

## Dataset
- Source: Cost of Living Index 2022
- Rows: Countries
- Key Features:
  - Cost of Living Index
  - Rent Index
  - Groceries Index
  - Restaurant Price Index
  - Local Purchasing Power Index

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook

## Analysis Performed
- Data inspection and cleaning
- Descriptive statistics
- Correlation analysis
- Identification of most expensive and cheapest countries
- Purchasing power comparison

## Key Insights
- Strong correlation between Cost of Living and Rent Index
- Some expensive countries have low purchasing power
- Several low-cost countries provide relatively higher affordability

## Conclusion
This analysis helps individuals, businesses, and policymakers understand global cost variations
and make informed decisions regarding relocation, investment, or policy planning.
